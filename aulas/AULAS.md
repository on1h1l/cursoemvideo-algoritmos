# Sobre as aulas

Os materiais desta pasta acompanham o **Curso de Algoritmos** do professor Gustavo Guanabara, disponível gratuitamente no site:

https://www.cursoemvideo.com/course/curso-de-algoritmos/

## Estrutura

- `pratica/` → exemplos demonstrados durante a aula
- `propostos/` → exercícios sugeridos para resolução pelo aluno

Os arquivos com sufixo `-ref` correspondem às versões fornecidas no material do curso.
